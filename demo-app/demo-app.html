Angular 2 Bootstrap 4 Dropdown

Simple directive with Universal Angular support. Uses CSS selectors to activate bootstrap dropdowns.

Install:
npm install ng2-bs-dropdown --save

Use:
import {DROPDOWN_DIRECTIVES} from 'ng2-bs-dropdown';
